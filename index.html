This page cannot read the CSTool API from github.io (or other sites) because the server does not send CORS headers for your origin. Refreshing cookies or opening CSTool in a hidden iframe does not change that — the response is still hidden from JavaScript here.

        
What works

        
    
          
  • Host this reader on the same origin as CSTool (then Fetch uses your session cookies automatically).
    • 
          
  • Or ask infra to allow Access-Control-Allow-Origin for your Pages URL on the CSTool API paths.
    • 
          
  • Or use Open CSTool below, run “Download & Analyze” there, then Choose log file here.
